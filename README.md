# вҡҷпёҸ Installation

### рҹ“Ҙ Clone Repository

```bash
git clone https://github.com/techiebella/notes.git
```

### рҹ“Ӯ Navigate to Project

```bash
cd notes
```

### рҹ“Ұ Create Virtual Environment

```bash
python -m venv venv
```

### в–¶пёҸ Activate Virtual Environment

**Windows**

```bash

```

**Linux / macOS**

```bash
source venv/bin/activate
```

### рҹ“ҡ Install Dependencies

```bash

```

### рҹӣ пёҸ Apply Migrations

```bash

```

### в–¶пёҸ Run Development Server

```bash

```
